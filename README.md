# Aplikasi To-Do List

## Deskripsi
Aplikasi sederhana berbasis PHP untuk mencatat tugas harian

## Fitur
- Tambah Tugas
- Tandai Tugas Selesai
- Hapus Tugas
- Ubah Tugas

## Struktur Folder
- index.php (halaman utama)
- style.css (styling)

## Cara Menjalankan
1. Clone repository
2. Salin folder ke htdocs
3. Jalankan XAMPP
4. Buka browser dan ketik "http://localhost/todolist"
